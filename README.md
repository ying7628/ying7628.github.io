# Man Smoking — Local Demo (folder: 1104_site)

這個資料夾包含一個簡單的靜態網頁，會顯示你放在同一資料夾的本機圖片 `man_smoke.jpg`。

檔案:
- `index.html` — 主頁，使用本機圖片 `man_smoke.jpg`。
- `styles.css` — 樣式檔案。

如何使用：
1. 把你的圖片放到此資料夾，並命名為 `man_smoke.jpg`。
2. 在 Finder 裡雙擊 `index.html` 開啟，或在終端輸入：

```bash
cd /Users/ying/Desktop/111407020/1104_site
open index.html
```

如果你還需要我幫你把現有圖片從其他位置複製到這裡，請告訴我圖片的完整路徑，我可以幫你搬或複製（或者教你用命令）。
